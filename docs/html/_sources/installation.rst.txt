
Installation
============

User Installation
-----------------
You can install MetronomeStatic via `pip`:

.. code-block:: shell

   pip install MetronomeStatic


Developer Installation
----------------------
To install for development, first clone the repository and then install
via pip's development mode.

.. code-block:: shell

   git clone <TODO>
   cd MetronomeStatic

   # Install the requirements
   pip install -r requirements.txt

   # Test the project
   pytest -s
