Universal AST (UAST)

UAST is a universal representation of ASTs across different languages. The UAST from MetronomeStatic comes from the project PLYJ, and modified for representing C/C++ ASTs.

The API document of UAST is HERE.

Here is a C file, assets/uast-demo.c:

1int main(int paramA, int paramB)
2{
3    if (paramA < 0)
4    {
5        return -1;
6    }
7    return paramA + paramB;
8}

To convert this file to UAST, just code like get_uast.py:

 1import json
 2import sys
 3
 4sys.path.append("../../..")
 5
 6from MetronomeStatic.clang_utils import *
 7from MetronomeStatic import *
 8
 9path = abspath_from_current_file("assets/uast-demo.c", __file__)
10
11"""
12Code above is critical for running this example,
13and not necessary when using in an interactive environment (IPython/Jupyter).
14
15↓↓ JUST FOCUS ON THE FOLLOWING CODE! ↓↓
16"""
17
18
19evaluator = ClangASTConverter()
20cursor = get_func_decl(
21    parse_file(
22        path,
23    ).cursor,
24    "main",
25)
26assert cursor is not None
27
28ret = evaluator.eval(cursor)
29
30with open("./outputs/dumped-ast.json", "w") as f:
31    json.dump(ret.to_dict(), f, indent=2, ensure_ascii=False)
